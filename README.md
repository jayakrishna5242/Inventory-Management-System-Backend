# 📦 InventoryPro

**InventoryPro** is a full-stack Inventory Management System built to manage products, stock levels, purchases, and sales efficiently through a modern dashboard.

## 🚀 Tech Stack

* **Frontend:** React + TypeScript + Vite
* **Backend:** Spring Boot (REST API)
* **Database:** MySQL
* **Containerization:** Docker

## ✨ Features

* Product CRUD management
* Purchase & sales tracking
* Automatic stock updates
* Low stock alerts
* Dashboard analytics (revenue, purchases, profit)
* Export reports

## ⚙️ Setup

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on **http://localhost:5173**
Backend runs on **http://localhost:8080**

## 🔗 API Base URL

```
http://localhost:8080/api
```
